# K-Desktop

[K-Desktop](https://github.com/Kobayashi82/K-Desktop) es una suite de productividad y automatización para `Windows` desarrollada en `Visual Basic .NET`. Permite ejecutar comandos, atajos globales, capturas de pantalla, grabación de audio y vídeo, menús contextuales personalizados y control remoto vía `UDP`.

En esta infraestructura se usa principalmente para ejecutar comandos en `Windows` de forma remota desde otros dispositivos de la red a traves de `UDP`.
