
# Portainer

Es una herramienta de administración para entornos de contenedores que proporciona una interfaz web para gestionar `Docker` de forma visual. Permite controlar contenedores, imágenes, volúmenes, redes y stacks sin necesidad de utilizar exclusivamente la línea de comandos.

En esta infraestructura se utiliza para centralizar la gestión del entorno `Docker`, facilitando la administración de todos los servicios desplegados. Desde `Portainer` se pueden desplegar nuevas aplicaciones, reiniciar servicios, consultar logs y supervisar el estado general del sistema.

Su función principal es simplificar la operación diaria del sistema `Docker`, especialmente en entornos con múltiples contenedores o servicios distribuidos, reduciendo la complejidad de la gestión manual por terminal.
